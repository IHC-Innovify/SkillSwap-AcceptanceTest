Pruebas de Aceptación del Proyecto SkillSwap

Este repositorio contiene las pruebas de aceptación para el proyecto SkillSwap (Innovify), desarrollado para el curso de IHC y Tecnologías Móviles (CC52) de la Universidad Peruana de Ciencias Aplicadas (UPC).

 ¿Qué es SkillSwap?

SkillSwap es una plataforma en línea diseñada para conectar a estudiantes de distintas universidades.
Su propósito es permitirles compartir e intercambiar conocimientos bajo un modelo de trueque, con la validación de tutores especializados en cada tema.

 Propósito de este Repositorio

Este repositorio aloja todos los archivos de pruebas de aceptación (Acceptance Tests) escritos en lenguaje Gherkin (archivos .feature).

El objetivo de estas pruebas es validar que el software cumple con los Criterios de Aceptación definidos en las 51 Historias de Usuario (User Stories) del proyecto, asegurando que las funcionalidades implementadas resuelven las necesidades de los tres segmentos objetivo:

Estudiantes que quieran aprender

Estudiantes que quieran enseñar

Coordinador Institucional

 Herramientas Utilizadas

Lenguaje de Pruebas: Gherkin (Sintaxis Given/When/Then)

Ejecutor de Pruebas (Test Runner): Cucumber, visual studio code.

Repositorio: GitHub
